## Backend API Routes
| Endpoint | Method | Description |
|---|---|---|
| `/api/wallet/balances` | GET | Moralis token balances (5 chains) |
| `/api/security/scan` | GET | GoPlus security scan (batched 10) |
| `/api/routes/best` | GET | LiFi cross-chain route optimization |
| `/api/ai/analyze` | POST | Gemini 2.5 Flash AI analysis |
| `/api/0g/store` | POST | 0G decentralized storage upload |
| `/api/0g/fetch` | GET | 0G report history / fetch by hash |

## Key Design Decisions
1. **WalletContext** provides `effectiveAddress` — all hooks take address as param (no hardcoded values)
2. **Demo mode** — falls back to Vitalik's address when no wallet connected
3. **0G failover** — if 0G unavailable, reports saved to localStorage with user notification
4. **Security scan** — batched GoPlus requests (10 at a time) to prevent server crash
5. **Swap history** — stored in localStorage (no backend needed)
6. **CSV export** — Portfolio page generates downloadable CSV
7. **PDF export** — Security page opens print dialog with formatted report

## Build Commands
- Frontend: `cd frontend && npx vite build`
- Backend: `npx next build`
- Dev: `npx next dev --webpack -p 3000` (backend) + `cd frontend && npx vite --port 5173` (frontend)

- No authentication/authorization (hackathon demo)
- No rate limiting on public API endpoints
- Frontend bundle is 619KB (could code-split)
